Integrates HTML Blocks and Google Maps modules. Modular architecture, enable only the modules you use.

== Description ==

Scale Lite Tools is a plugin that embeds modular plugin components. Enable in settings only the modules you use.

= HTML Block =

* Creates HTML Block custom post type
* Creates widget that can be used with shortcode [html_block id="200"]

= Google Maps =

* Adds Google Maps v3 with abillity to place multiple markers on generated map via shortcode. You will have to generate your own Google Maps API Key.

= It works with any theme =

Modules are designed as custom post types and work with any Wordpress installation starting with version 4.7.
Just enable the modules you want to use on the settings page and you are ready to go.

== Installation ==

This section describes requirements,how to install the plugin and plugin features.

= Requirements =

* Wordpress version: any released version starting with 4.7
* PHP version: ideally version 7.0.x as the plugin is tested on this version and above

= How to install =

1. Search and install the `Scale Lite Tools` plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the `Settings->Scale Lite options` screen to activate or deactivate modules.
4. Active modules are listed under the dashboard. Plugin folder is `scale-lite-tools`.
